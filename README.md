# Superstore Retail Analytics & Sales Optimization

An end-to-end Exploratory Data Analysis (EDA) and Statistical Analysis project conducted during my Data Science Internship.

## 🚀 Project Phases & Methodologies
* *Data Cleaning:* Streamlined raw data, handled missing values, and standardized decimal formats.
* *Category Performance:* Built interactive pivot tables to audit sales vs. profit margins across product lines.
* *Time Series Analysis:* Modeled seasonal variations and quarterly sales growth using customized combo charts.
